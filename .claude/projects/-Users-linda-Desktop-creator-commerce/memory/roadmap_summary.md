## 战略定位

"我们是设计师的出版社，不是图书馆，不是印刷厂。"
- 区别于 Redbubble（流量平台）和 Printify（供应链工具）
- 定位：创作者商业化合伙人，利益深度绑定，对结果负责
- 优势：数据+人工主动选品、投放+设计师受众+社媒三叠加流量、中美双供应链

## MVP 核心原则：人工先跑通，不做自动化

MVP 阶段（5-6月）聚焦三个问题，全部人工解决：

### 问题1：设计师不知选什么产品
- **MVP**: 运营团队人工选品，不让设计师自己选
- 阶段一(3-6月后): 规则引擎半自动推荐
- 阶段二(6-12月后): AI Agent 选品预测

### 问题2：图案文件质量无把关
- **MVP**: 人工逐一检查 DPI、颜色模式、尺寸
- 阶段一: 半自动检测脚本
- 阶段二: 全自动检测 + AI 修复建议

### 问题3：Mockup 不满足销售需求
- **MVP**: 双轨制（预览版 vs 广告/销售版分开管理）
- 阶段一: 标准广告素材模板库
- 阶段二: AI 自动生成广告级 mockup

## 选品决策记录表（最重要的数据资产）

前 50 条人工填写，是 AI Agent 体系的数据地基。

### 3.1 图案基础信息
| 字段 | 说明 |
|------|------|
| 图案ID | 系统自动生成 |
| 设计师ID | 签约设计师标识 |
| 上传日期 | 图案进入系统日期 |
| 图案名称 | 设计师/运营命名 |
| 风格标签 | 多选：极简/波普/插画/文字类/自然系/暗黑系/复古/卡通/几何 |
| 主色调 | 影响产品颜色匹配 |
| 目标人群判断 | 年龄段、性别偏向、兴趣标签 |
| 图案文件质量 | 合格/需修改/退回 |
| 质量问题备注 | 具体问题类型 |

### 3.2 选品决策
| 字段 | 说明 |
|------|------|
| 推荐产品品类（第一优先） | 主推产品 |
| 推荐产品品类（第二优先） | 备选产品 |
| 推荐理由 | 数据依据或经验判断 |
| 不推荐品类及原因 | 明确排除 |
| 定价建议 | 售价区间 |
| 决策人 | 谁做的选品决策 |
| 决策日期 | 完成日期 |

### 3.3 上架运营追踪
| 字段 | 说明 |
|------|------|
| 上架日期 | 正式上架日期 |
| 上架平台 | 独立站/设计师Shopify/Etsy |
| 首月浏览量 | 页面浏览量 |
| 首月加购率 | 加购/浏览 |
| 首月转化率 | 成交/浏览 |
| 首月GMV | 实际成交金额 |
| 30天复盘结论 | 继续主推/调整定价/更换品类/下架 |
| 最终状态 | 活跃/暂停/下架 |

## 开发路线图

| 阶段 | 时间 | 核心任务 | 不做什么 | 里程碑 |
|------|------|---------|---------|--------|
| MVP | 5-6月 | 人工跑通全流程 | 不做AI、自动化、设计师自助工具 | 8+设计师, 50+SKU, 月GMV $3K+ |
| 阶段一 | 7-9月 | 半自动化 | 不做AI训练 | 选品准确率>70%, 返修率<20% |
| 阶段二 | 10月+ | AI Agent | 数据不够不启动 | AI上线, 人工干预减少50% |

## 6月底 Go/No-Go 检查点

| 指标 | 最低达标线 |
|------|-----------|
| 签约设计师 | ≥ 8 |
| 上架SKU | ≥ 40 |
| 月GMV | ≥ $3,000 |
| 单设计师最高月GMV | ≥ $500 |
| 社媒自然播放 | ≥ 5,000次 |
| 文件返修率 | < 30% |

全部达标 → 进入阶段一。任意核心指标严重不达标 → 暂停投放，重新评估。
